メンバー抽選プログラムの使いかた

簡単ver.
以下のコマンドをターミナルに入力してください
$ gcc -o member member_choose.c
$ ./member

あとは、言われた通りに入力するだけです。
基本的には半角・小文字入力ですが、名前は日本語にも対応しています。


詳細ver.
member_choose.c を適当にコンパイルしてください
それを実行してください。

内容としては
A.抽選の男女分けの有無(男女分けしても男女無しにすることもできます。)
B.参加人数(男女分けの場合それぞれの人数)
C.当選人数(男女分けの場合、男・女・男女問わない当選人数)
*A,Cは半角・小文字で入力してください。

開発環境は以下の通り(実行環境も同様)
コンピュータ MacBook Air (13-inch, Early 2015) OS macOS Mojave ver. 10.14
プロセッサ 1.6 GHz Intel Core i5
メモリ 4 GB 1600 MHz DDR3
Visual Studio Code  バージョン1.30.0 (1.30.0)

ただ、シンプルなプログラムなので大体の環境で動くと思います。
windowsはコンパイラは搭載されてないので、適当にインストールしてください。
